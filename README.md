# Restaurant Management System - Backend
This is a collaborative project developed as part of a team. The system focuses on managing restaurant operations, customers, and security.

## Tech Stack
*   **Backend:** ASP.NET Core Web API
*   **Database:** SQL Server, Entity Framework Core (Code First)
*   **Authentication:** JWT, ASP.NET Identity
*   **Tools:** Postman, Git

## My Responsibilities
I was responsible for the core Backend development, including:
*   **User Management:** Implementing Authentication & Authorization using JWT.
*   **Security:** Developing the **OTP Recovery** system via Email for password resets.
*   **Business Logic:** Building RESTful APIs for **Customer** and **CustomerRank** modules (CRUD operations).
*   **Database Design:** Designing schemas and handling migrations with EF Core.
